# lstm_multi_modal_UNet
Implementation of UNet, multi-modal U-Net, LSTM multi-modal Unet for BRATS15 dataset with Pytorch


## Prerequisites
* Python >= 3.6  
* Pytorch >= 1.0.0  
* SimpleITK   
* numpy
* scipy
